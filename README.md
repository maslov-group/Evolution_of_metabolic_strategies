## Environmental fluctuation shapes the evolution of metabolic strategies

Repository for all simulations, analysis and visualization code in the project examining how environmental factors (resource supply ratio fluctuations and resource quality difference) drives the evolution of metabolic strategies (e.g., specialization, hierarchical utilization and co-utilization).

| **Folder/File**                                              | **Description**                                              |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **`phi_pre_evol_with_leaky_mod.py`** and **`phi_pre_evol_with_leaky_mod.sh`** | Code for the numerical experiments.                          |
| **`data/`**                                                  | Directory where raw data are stored.                         |
| **`leaky_diaux_observer_v2.ipynb`**                          | Includes all code for generating the analysis and visualizations found in the main text figures of the manuscript. |
| **`leaky_diaux_supp_figs_v1.ipynb`**                         | Contains code for all analysis and visualizations used in the **supplementary figures.** |
|                                                              |                                                              |
